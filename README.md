# Corrida Kart
Programa calcula o resultado de uma corrida de kart com base em um arquivo de log_corrida.txt

```
Kotlin 1.3.41
```
* Executar todos os comandos na raiz do projeto em um terminal

## Build
```
mvn clean package
```

## Executar Aplicação
```
java -jar target/corridakart.jar
```

## Executar Tests
```
mvn test
```
